# Neco Website Features

## Overview

A modern, professional website for an IT agency with multilingual support, dark/light themes, and comprehensive sections to showcase services, pricing, and portfolio.

## Core Features

### 1. Multilingual Support (i18n)
- **Languages**: English, Russian, Uzbek
- **Implementation**: next-intl for seamless translations
- **Language Switcher**: Easy-to-use dropdown in header
- **SEO**: Proper hreflang tags for each language

### 2. Theme System
- **Modes**: Light and Dark themes
- **Persistence**: User preference saved across sessions
- **Smooth Transitions**: Elegant theme switching animations
- **System Detection**: Respects user's system theme preference

### 3. Responsive Design
- **Mobile First**: Optimized for all screen sizes
- **Breakpoints**: sm (640px), md (768px), lg (1024px), xl (1280px)
- **Touch Friendly**: All interactive elements sized for touch
- **Tested**: Works perfectly on phones, tablets, and desktops

### 4. Performance Optimizations
- **Fast Loading**: Optimized images and code splitting
- **Lazy Loading**: Sections load as you scroll
- **Smooth Animations**: Framer Motion for performant animations
- **SEO Optimized**: Fast Core Web Vitals scores
